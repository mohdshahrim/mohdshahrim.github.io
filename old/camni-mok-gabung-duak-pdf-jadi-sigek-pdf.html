Camni mok gabung duak PDF jadi sigek PDF?

Leave a reply

Cara tuk untuk dalam komputer jak, bukan untuk telepon android/apple

Katakan kitak ada duak fail PDF (macam dalam gambar dibah ya)

Kitak tek nak gabong duak igek PDF ya jadi sigek PDF jak, tanpa mengganggu kualiti. Jadi, apa yang kitak perlu polah ialah 3 langkah tuk:


1. Download tool nama merge2pdf (boleh skip mun dah ada)
– download dari link tuk https://github.com/ajaxray/merge2pdf/releases/download/v1.1.0/merge2pdf_windows-386.exe
– engkah tool merge2pdf ya ke satu folder (dekat Desktop pun boleh)


2. Engkah PDF kitak dalam satu folder dengan merge2pdf tek
– tuk untuk memudahkan kitak nampak proses tuk dari sudut yang lebih jelas
– kamek rename “merge2pdf_windows-386.exe” ke “merge2pdf.exe” untuk merik senang


3. Bukak command prompt, taip command macam dibah
merge2pdf.exe final.pdf depan.pdf belakang.pdf
– mun sik pasti, cuba ikut step yang kamek tunjuk dibah tuk (gif)
– waktu dalam folder ya, tekan Alt + D
– pasya taip “cmd”, Enter
– bila Command Prompt dah keluar, taip terus nak dibah tuk
merge2pdf.exe final.pdf depan.pdf belakang.pdf

4. Selesai
– fail PDF yang baru akan muncul sama dalam folder tokrang molah tuk tek


Cara tuk sik sesuai untuk orang yang sik biasa make command prompt, tapi kamek galakkan kitak orang belajarlah make command prompt.

Terima kasih



Rujukan:
