5 software yang tokrang patut ada dalam PC

Leave a reply

Tapi sebelum ya…
kandungan post tuk adalah 100% pendapat peribadi
jadi nya sik semestinya betul
dan nya sik semestinya sesuai untuk semua orang
majoriti tuk untuk Windows jak, bukan untuk android/ios/mac

Sebagai nota ringkas:
1. Notepad++
2. Everything (file search)
3. SingleFile
4. Greenshot
5. SumatraPDF Reader

#1 Notepad++

“aik, padahal tiap PC dah ada Notepad”
yes, tapi Notepad++ boleh adjust tema warna
ada syntax highlighting (cun bagi orang yang edit code atau special document)
senang gik mok molah carian teks
setiap file dalam sigek tab (senang ngurus banyak file serentak)

lagipun, Notepad yang biasa sik nunjuk mana-mana file text nak pernah dibukak, dan yang paling best ialah, Notepad++ sik menghilangkan apa-apa teks nak ditulis kita andai kata kita sik sengaja tertutupnya dan sik sempat save…

Laman web rasmi: https://notepad-plus-plus.org

#2 Everything (file search)

“aik, bukan nang dapat search dari Start menu kah?”
yes, tapi mun mok search file atau folder, Everything lebih cekap
performance laju dan softwarenya pun lightweight
dapat search guna wildcard, contoh mun file mp4, taip jak *.mp4

Laman web rasmi: https://www.voidtools.com

#3 SingleFile

sebenarnya tuk bukan software, tapi Browser Extension (Chrome, Firefox..)
nya dapat save page website menjadi sigek file html
file html ya, bila dibukak, masih mengekalkan sifat page ya tadik (walaupun offline)

Mun kita save page dengan cara biasa, page dari file html ya kelak selalunya jadi cacat… pasya kadang-kadang bila dibukak waktu offline, pande rusak terus… dengan SingleFile, insyaAllah sekda masalah.

Laman web rasmi:
https://github.com/gildas-lormeau/SingleFile
atau
https://chrome.google.com/webstore/detail/singlefile/mpiodijhokgodhhofbcjdecpffjipkle

#4 Greenshot

Greenshot dapat tolong kita ambik screenshot dan auto save
dapat setting destination untuk tempat screenshot disave
format dapat pilih png/jpg/bmp
dapat pilih screenshot seluruh screen, window tertentu atau custom
nya boleh ada effect bunyi kamera

Memang Windows udah ada utiliti screenshot nya dikpun, dan ada Snipping Tool, tapi Greenshot tetap lighweight walaupun padat dengan ciri-ciri dan setting yang power. Ohh by the way, tolong jangan sesekali make Lightshot (sejenis software sama macam Greenshot juak)

Laman web rasmi: https://getgreenshot.org

#5 Sumatra PDF Reader

pake bukak PDF + tersangat lightweight
buang jak AdobeReader ya, guna tuk bagus gik

Laman web rasmi: https://www.sumatrapdfreader.org/free-pdf-reader

So ya lah 5 software yang Laksamana galakkan tokrang guna dalam PC untuk memudahkan kerja dan meningkatkan produktiviti… tapi lain orang lain cara so sik perlulah ikut 100%… dan yang penting, semua ya adalah FREE

Sebagai nota ringkas:
1. Notepad++
2. Everything (file search)
3. SingleFile
4. Greenshot
5. SumatraPDF Reader

Have a nice day
