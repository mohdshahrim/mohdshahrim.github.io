Snippets

snippets of command or code that I considered useful

cmd = Windows Command Prompt

terminal = Linux Terminal


Search files faster (cmd)

dir "\search term*" /s

List out running Services (cmd)

net start

Generate hash of a file (cmd)

certutil -hashfile "C:\Users\Administrator\Desktop\tehee.txt" sha256

Find system boot time (cmd)

systeminfo|find "Boot Time"

Install specific Laravel (cmd)

composer create-project laravel/laravel="8.6.12" myproject

Echoing text to file without quotes (cmd)

echo|set /p="Hello" > test.txt

Displaying current directory (cmd)

echo %cd%
